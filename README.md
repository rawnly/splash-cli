## Installation
To install `splash-cli` you must use a **node package manager** such as [yarn](yarn) or [npm](npm).


```bash
	$ npm i -g splash-cli
```

## Usage
![usage](https://cloud.githubusercontent.com/assets/11269635/21428079/7b24cc80-c858-11e6-8dc3-2e164d23804a.gif)
